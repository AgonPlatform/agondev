### Example of Large Buffered RGBA2222 Bitmaps

Loads a large colour bitmap's data from file into a buffer in chunks. 

Displays bitmap on screen using the Plot Bitmap function.

Once loaded into vdp, the image can be manipluated very quickly with just a few commands.

Rotate or flip image, horizontally or vertically.

Press ESC to exit the program.


Image is loaded from file as raw binary data. 
Ensure sample image file 'boat.RGB2' is placed in the same folder as the binary application
